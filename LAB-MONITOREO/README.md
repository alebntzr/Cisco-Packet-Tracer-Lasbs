# Laboratorio - Monitoreo

## Objetivo
Monitorizar el tráfico utilizando un servidor SYSLOG, un servidor FTP y un sniffer

## Tecnologías
- Cisco Packet Tracer
- Servidor FTP
- Servidor SYSLOG
- Routers
- Sniffer

## Configuración
- Servidor FTP
- Sniffer
- PC'S
- Servidor SYSLOG
- Simulacion

## Verificación
Se comprobó el correcto funcionamiento del archivo enviado al servidor FTP y detectando sus vulnerabilidades en el username y el password del dispositivo de origen. Se comprobó el monitoreo de tráfico provenientes de todas las PC's utilizando el sniffer y el servidor SYSLOG, detectando a traves del registro de que red proviene y que dispositivos hicieron ping en el router dentro de la red del sniffer.
