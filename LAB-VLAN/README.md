# Laboratorio de VLANS

# Objetivo
Configurar VLANS y permitir la comunicación mediante un enlace Trunk

# Tecnologías
Cisco Packet Tracer
Switch Cisco 2960

# Configuracion
VLAN 2
VLAN 4
VLAN 9
Trunk 802.1Q

# Verificacion
Se comprobó la comunicación mediante pings entre equipos pertenecientes a la misma VLAN y se comprobó también la incomunicación de dispositivos fuera de la VLAN respectiva
